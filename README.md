# church_database
 A church database system that helps your catholic church and its congregation to manage the information and activities of the church oon a day to day basis. The features are;

 1. Users Management: You can create Admin users and members of your congregation can register as needed.
 2. Organizations and Societies: You can create Organizations and Societies that exists in the church and the members can can be added.
 3. Sacraments: All members can record sacarements taken as done on batismal cards.
 4. Donations and Tithes: All members can Make Donations and pay tithes to the church using Paystack payment gateway
 5. Mass booking: You can book masses online as you choose to.
 6. Manual Payments: Admins can record all other transactions that occur outside of the system.
 7. Feeds/Annoucements: Admins can make annoucements on the system and their congregation can view them.
 8. Calendar: All events in the church can be recorded as needed on the system. And everyone can view and access the calendar as needed.
